# graduation_project_hyonnnn

■サービス概要
妊婦さんが毎日前向きに過ごせるようサポートするやる気スイッチアプリです。なかなか行動に移せない妊婦さんに対し、アプリ側から日々のタスクやアクションを提案し、ポジティブなメッセージで気分を上げます。
モチベーションを維持し、安産に向けて健康的な生活習慣を自然にサポートします。

■ このサービスへの思い・作りたい理由
私は現在臨月の妊婦です。臨月に入ると、安産のためには「運動をしたほうがいい」「トイレ掃除が効果的」など助産師さんからアドバイスを受けます。しかし、なかなかやる気が起きず、「明日運動したらいいか」と安産のために行動することを後回しにしてしまいます。そんな体験から、お腹の中にいる赤ちゃんのためにも、少しずつでもやる気を出して体を動かせるようサポートするアプリを作りたいと思うようになりました。このサービスは、妊婦さんが無理なく続けられ、心身ともにリラックスして生活できることを目指します。

■ ユーザー層について
出産準備が本格化する時期に、日々の行動を促すアプリの価値が特に高くなる臨月妊婦さんを対象とします。

■サービスの利用イメージ
ユーザーはアプリを開くと、「今日やること」が選択形式で表示されます。
選択した行動を実践するとポジティブメッセージが表示され、達成感や前向きな気持ちが得られます。
小さな達成体験を積み重ねることで、心身ともに健康的に過ごせるとともに、母親になる自覚が芽生え、ポジティブな気分で赤ちゃんの誕生を迎えることができます。

■ ユーザーの獲得について
妊婦向けSNSでのプロモーション。
妊婦健診の問診時に助産師さんを通じての紹介。

■ サービスの差別化ポイント・推しポイント
妊婦向けアプリの多くは情報提供や記録に特化していますが、モチベーション維持に焦点を当てたアプリは少ないです。「安産のために体を動かさないといけない」と考えると自然と脳へ負荷がかかるため、アプリ側から「今日やること」を選択形式で提案し、いつの間にか体を動かしていた＝安産につながるといった小さな達成を経験できることが推しポイントです。達成時には、ポジティブメッセージを表示することで、自然に健康的な生活習慣をサポートし、臨月妊婦さんの心理的負担を軽減にも繋がります。

■ 機能候補
MVPリリース
1.ユーザー登録 / ログイン
2.毎日の「やること」表示
3.ポジティブメッセージの表示
4.行動チェック機能（達成・未達成の記録）

本リリース
5.達成履歴の振り返り機能（カレンダーやグラフ表示）
6.家族やパートナーと共有できる応援機能

■ 機能の実装方針予定
MVPリリース
1.ユーザー登録 / ログイン:Deviseを使用したユーザー認証とセッション管理
2.毎日の「やること」表示:CRUD機能・モデルとビューの連携
3.ポジティブメッセージの表示:JavaScript,データベースに登録したメッセージをランダムに抽出して表示
4.行動チェック機能（達成・未達成の記録）:CRUD操作

本リリース
5.達成履歴の振り返り機能（カレンダーやグラフ表示）:Chart.jsを用いて達成状況をグラフ化
6.家族やパートナーと共有できる応援機能:ユーザー間の関連付けを活用
